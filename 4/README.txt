Нужна БД (делала на MS SQL Server)

В классе ComputerService изменить строку подключения.
====================

UI
Все сам подгрузится при запуске, если сервер БД поднят. 

Добавление
Ввести значения в 3 поля и нажать "Add"
"Type" выбрать из дропдауна 

Удаление
Выбрать строку в таблице
Клик по "Delete"

Редактирование
Клик по нужной строке
Ввести новые данные в 3 поля
Клик по "Edit"

"Load" подгружает данные из бд( позволяет обновить таблицу, если несколько клиентов одновременно добавляют записи)
